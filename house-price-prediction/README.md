## 🏠 House Price Prediction

This machine learning app predicts the **price of a house** based on key features such as:

* Living Area (sq ft)
* Number of Bedrooms
* Number of Bathrooms

The model was trained using **Scikit-learn** and deployed as an interactive **Streamlit web application**.

### 🧠 Model Details

* Algorithm: **Regression Model (Scikit-learn)**
* Task: **Predict house prices**
* Output: Estimated house value in USD
