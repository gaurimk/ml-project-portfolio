# 🏠 House Price Prediction

A Machine Learning web application that predicts house prices based on property features.

## 🚀 Live Demo

👉 https://ml-project-portfolio-grmeeh8kfruqdaftxjawp7.streamlit.app/

## 📊 Project Overview

This project uses a machine learning model trained on housing data to estimate house prices.
Users can enter house features and get an instant price prediction.

## 🧠 Model

* Algorithm: Linear Regression
* Framework: Scikit-learn
* Model saved using **joblib**

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## ▶️ Run Locally

Clone the repository:

```
git clone https://github.com/gaurimk/ml-project-portfolio.git
```

Navigate to project:

```
cd house-price-prediction
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the Streamlit app:

```
streamlit run app.py
```
