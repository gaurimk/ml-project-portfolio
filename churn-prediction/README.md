## 📊 Customer Churn Prediction

This project predicts whether a telecom customer is likely to **churn (leave the service)** or **stay**, based on customer attributes such as tenure and monthly charges.

### 🧠 Machine Learning Model

* Algorithm: **Random Forest Classifier**
* Task: **Binary Classification (Churn / Stay)**
