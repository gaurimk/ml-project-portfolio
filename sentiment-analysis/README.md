### 🎬 Movie Review Sentiment Analysis

[![Open App](https://img.shields.io/badge/Live%20App-Streamlit-red?style=for-the-badge&logo=streamlit)](https://ml-project-portfolio-owz2cuq7cvt9oynu7wqxqx.streamlit.app/)

This NLP project predicts whether a movie review is **positive or negative** using **TF-IDF vectorization and a machine learning model**.

**Features**
- Text preprocessing
- TF-IDF vectorization
- Sentiment classification
- Streamlit interactive interface

**Tech Stack**
- Python
- Scikit-learn
- NLP
- Streamlit
